Portafolio

A continuación se listan nuestros proyectos mas recientes y se da una breve explicación sobre la implementación.



Sistema de control de órdenes de servicio para empresas de reparación celular


Sistema Web y Software como servicio que se le renta a empresas cuyo giro radica en la reparación de cualquier tipo de dispositivo, con el cual llevan el control desde la captura hasta la entrega de dicho dispositivo, pudiendo pasar y registrando los status, ya sea en reparación, falta de refacciones, terminado o cancelado.



Portal web de OutdoorSonora


Propuesta web, diseño, maqueteo web, front-end y back-end. Sistema de administración de contenido que controla entradas tipo noticias, galerías multimedia (video y fotos), calendario de eventos, blogs, reseñas de figuras publicas y publicidad. El administrador de contenido tiene como objetivo alimentar el portal web de la misma empresa con las entradas mas actuales, ubicándolas en su secciones correspondientes llevando una paginación del historial de las mismas.



Portal web de Ecity


Maqueteo web, front-end y back-end. Sistema de cupones para empresas las cuales son categorizadas, al tener cupones disponibles este gestiona el proceso de canje, involucra envió de mensajes de texto a través de internet a teléfonos móviles, comunicación con el cliente a través del correo electrónico y un validador de cupones activos.



Sistema para generar encuestas y distribuir vía e-mail


• Permite la creación de encuestas asignándole un nombre, descripción, fecha de cierre e indicando si está habilitado o no. • Para cada reactivo o pregunta se indica si la respuestas será requerida para contestar y el tipo: abierta (texto, número), de múltiple o única selección. • Las encuestas se le asignan a un grupo de interés para la distribución, permitiendo que llegue a través de e-mail a cada integrante del grupo, lo que permite que la encuesta sea contestada solo una vez y poder conocer lo que contestó. • Reportes por encuesta y por pregunta.



Portal web de OlaSonora


Propuesta web, diseño, maqueteo web, front-end y back-end. Sistema de administración de contenido que controla la alta de entradas tipo noticias, gestión de lectores, reporteros, capturistas y administradores. También incluye un administrador de banners, integración con redes sociales y uso de la api de YouTube para el control de eventos en la reproducción de videos.



Portal web de Dinamica Industrial SA de CV


Propuesta web, diseño, maqueteo web, front-end (Jquery). Sitio web informativo y de contacto soportado por dispositivos móviles.



Tienda en linea de Megacomm


Propuesta web, diseño, maqueteo web, front-end (JavaScript & Jquery), implementacion de API de CVA (PHP). Tienda en linea la cual consume una API externa que filtra productos por categoría, marca, número de parte y/o descripción. Además es compatible con dispositivos móviles.
